## 芯片选择

- `arm64`：适用于 Apple Silicon 芯片 Mac，例如 M1、M2、M3、M4
- `x64`：适用于 Intel 芯片 Mac

判断方法：
- 打开 `苹果菜单 > 关于本机`
- 如果看到 `Apple` 芯片型号，下载 `arm64`
- 如果看到 `Intel`，下载 `x64`

## 安装步骤

1. 下载与你设备对应的 `.dmg`
2. 打开安装包
3. 把 `AutoBoss` 拖入 `Applications`
4. 到 `Applications` 中打开 `AutoBoss`

如果首次打开被系统拦截：

1. 打开 `系统设置 > 隐私与安全性`
2. 找到与 AutoBoss 相关的安全提示
3. 点击允许
4. 再次打开应用

## 首次使用建议

首次进入应用后，建议按这个顺序操作：

1. 先完成环境检查
2. 确认 Google Chrome 已安装且窗口可见
3. 根据应用提示授予必要权限
4. 再继续后续配置和使用
