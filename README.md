# AutoBoss

AutoBoss is a closed-source macOS desktop application distributed through GitHub Releases.

This repository provides:
- Installer downloads for macOS
- Basic installation and usage documentation
- Release notes

Source code is not published in this repository.

## Downloads

Download the latest installers from the Releases page:

- Apple Silicon: `AutoBoss-1.0.0-arm64.dmg`
- Intel Mac: `AutoBoss-1.0.0-x64.dmg`

Direct release page:
- [Releases](../../releases)

## Which Build Should I Choose

- `arm64` is for Apple Silicon Macs such as M1, M2, M3, and M4.
- `x64` is for Intel-based Macs.

## Installation

1. Download the correct `.dmg` for your Mac.
2. Open the disk image.
3. Drag `AutoBoss` into `Applications`.
4. Launch the app from `Applications`.
5. If macOS blocks the app on first launch, open:
   `System Settings > Privacy & Security`
   and allow the app to run.
6. Follow the in-app prompts to grant the required permissions.
