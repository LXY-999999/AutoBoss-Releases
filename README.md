# AutoBoss
[English](./README.en.md) | [中文详细说明](./README.zh-CN.md)
一个Boss直聘自动化工作投递助手。
## 它能帮助你完成
1. 支持应聘者每天0花费自动投递120份工作（boss的每日投递上限是150，留30个自投的空间），只要坚持每工作日投，我自己是每周会有稳定2-3个约面
2. 支持多城市、多岗位筛选，然后可以设置每个岗位投递的数量
3. 不支持直接投递简历，这个原因很简单，投递岗位（打招呼）可以自动化，然后hr会问你要简历，但是真正要不要去这个岗位的决定权在我们自己手里，因此决定是否投递简历，也是我们初步筛选对方的一个过程

## 我该下载哪个版本

- `arm64`：适用于 Apple Silicon 芯片 Mac，例如 M1、M2、M3、M4
- `x64`：适用于 Intel 芯片 Mac

快速判断方法：
- 打开 `苹果菜单 > 关于本机`
- 如果看到 `Apple M1/M2/M3/M4`，请选择 `arm64`
- 如果看到 `Intel`，请选择 `x64`

## 安装步骤

1. 下载对应的 `.dmg` 文件
2. 打开安装包
3. 将 `AutoBoss` 拖入 `Applications`
4. 在 `Applications` 中打开 `AutoBoss`
5. 如果首次启动被 macOS 拦截，请前往：
   `系统设置 > 隐私与安全性`
6. 允许应用运行后，再次打开

## 首次打开说明

首次打开时，AutoBoss 可能会请求本地自动化所需的权限。

通常你需要：
- 确保 Google Chrome 已安装并可见
- 按照应用内环境检查逐项完成
- 在 macOS 中授予所需权限
- 返回应用重新检查环境

## 常见问题

如果应用打不开：

1. 打开 `系统设置 > 隐私与安全性`
2. 滚动到页面下方的安全提示
3. 允许 `AutoBoss` 运行
4. 然后重新打开应用

如果下载错了版本：
- Apple Silicon Mac 请下载 `arm64`
- Intel Mac 请下载 `x64`
